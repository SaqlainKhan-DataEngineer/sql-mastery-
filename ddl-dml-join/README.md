# SQL Joins Assignment

This project demonstrates the use of **SQL DDL, DML, and Joins**.  
It contains a small database with `customers`, `product`, `orders`, and `orderdetail` tables.

---

## 📂 Project Structure
- **SQL-Joins-Assignment.sql** → Contains table creation, data insertion, and queries.

---

## 🗄️ Database Schema
- **customers** → Stores customer information.  
- **product** → Stores products and their prices.  
- **orders** → Stores customer orders.  
- **orderdetail** → Stores which products were ordered and their quantities.  

---

## 📝 Queries Implemented
1. **INNER JOIN** – Show customers with their orders.  
2. **LEFT JOIN** – Show all customers and their orders (including those without orders).  
3. **LEFT JOIN + NULL** – Show products never ordered.  
4. **JOIN multiple tables** – Show order details with customer name, product, quantity, and total price.  
5. **GROUP BY** – Show number of orders per customer.  
6. **ORDER BY + LIMIT** – Show top 2 spenders.  
7. **Popular product** – Find the most frequently ordered product.
