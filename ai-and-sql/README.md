## AI and SQL

### Copilot vs ChatGPT
- Copilot: Lives inside your editor, 
  suggests code as you type
- ChatGPT: Separate tool, better for 
  explaining concepts

### How to Prompt AI for SQL
- Bad: "write a query"
- Good: "Write a SQL query to find top 
  5 customers by revenue in 2024 
  using AdventureWorksDW2022"

### Key Learning
The better your prompt, the better 
the output. AI is a tool — not a replacement.
